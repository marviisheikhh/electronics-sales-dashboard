# Electronics Sales Dashboard (Excel)

An interactive Excel dashboard designed to analyze and visualize electronics sales data across various regions and product categories. 

## Features
- **KPI Cards:** Track Total Sales (Rs. 5,541,500), Total Orders (20), Average Order Value (Rs. 277,075), and Best Selling Category (Mobile).
- **Dynamic Slicers:** Interactive filtering by Region and Product Category.
- **Data Visualizations:** Column charts visualizing sales distribution by region and category.

## File Structure
- `electronics-sales-dashboard.xlsx` - The core dataset, pivot tables, and dashboard.
- `dashboard 1.PNG` - Preview of the final layout.

## How to View
To interact with the slicers, download the `.xlsx` file and open it in Microsoft Excel.# electronics-sales-dashboard
